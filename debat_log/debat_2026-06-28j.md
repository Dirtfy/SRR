# Debat Log — 2026-06-28j

**Debator**  
**Scope:** Debug icon yellow border + release signing config

---

## Findings

**W1 — Release signingConfig uses the debug keystore**

`signingConfigs.getByName("debug")` references the auto-generated debug keystore
(`~/.android/debug.keystore`, password "android"). APKs signed with this key cannot
be published to the Play Store and have a known public key. Acceptable for development
testing; must be replaced with a real keystore before shipping to end users.

**M1 — Yellow border in the adaptive icon is a rectangle (not rounded)**

The four rectangular path segments in `ic_launcher_background.xml` form a sharp-cornered
frame. Adaptive icons are clipped to a shape (circle, squircle) by the launcher, so the
sharp corners are usually not visible. No practical impact.
