# Debat Log — 2026-06-28i

**Debator**  
**Scope:** Account label in top bar + hardware back button navigation

---

## Findings

**W1 — Email read outside Compose recomposition scope**

`AccountLabel()` uses `remember { Firebase.auth.currentUser?.email }`. This snapshot
is captured once per composition and won't update if auth state changes mid-session.
Acceptable: auth state changes (sign-in/out) always navigate to a new fragment, so the
label is always freshly composed when it matters.

**W2 — systemBackCallback left disabled after fallback to system back**

When `onBackClick()` disables the callback and hands off to `onBackPressedDispatcher`,
the callback stays disabled. If the activity somehow survives (edge case), the callback
won't intercept back again. In practice the activity finishes so this is harmless.
Acceptable for prototype.

**M1 — Column in RowScope actions may expand the AppBar height**

`Column { AccountLabel(); IconButton }` stacks email (~12dp) above icon (48dp) = ~60dp
inside a 64dp AppBar. Works, but leaves very little breathing room. No runtime error
expected since MD3 clips internally.
