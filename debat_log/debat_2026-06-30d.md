## Changes Reviewed

2. `RemoteStorageRepository` — implemented via `Firebase.storage.getReferenceFromUrl(url).delete()`
3. `storage.rules` — split `write` into `create, update` (with size guard) + `delete` (auth only); fixes silent delete deny due to `request.resource` being null on delete

6. `UserViewModel.onDismissEditItemImageDialog` — if dialog had an `imageUrl` (uploaded but not saved), deletes it from Storage on dismiss
