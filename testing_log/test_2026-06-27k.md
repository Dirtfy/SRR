## Investigation Notes

### Prior failures (fixed before this log)

**UI tests (15/16 `UserScreenUiTest`)** — "No compose hierarchies found"  
Cause: Stale compiled class files from incremental build after large UI changes.  
Fix: `./gradlew clean` + full rebuild.  
Verdict: Not a code defect; build hygiene issue.

**Firebase tests (13 failures)** — Documents persisting between tests  
Root cause: `deleteUrl()` called `connect()` + `disconnect()` without reading `responseCode`. Java's `HttpURLConnection` (backed by OkHttp on Android) is lazy — it does not actually send the HTTP request body until the response is triggered. The emulator never received the DELETE, so documents from test N accumulated into test N+1.  
Fix: Changed `conn.connect()` → `conn.responseCode` in all three test files.  
Verdict: Fixed. All uniqueness and isolation tests now pass.
